## Local Development

### Prerequisites

- Node.js (LTS recommended)

### Install

```bash
npm install
```

### Run

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Deployment

This is a standard Vite app; you can deploy it to services like Vercel, Netlify, or GitHub Pages.
